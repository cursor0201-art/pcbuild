## 🚀 Специальные функции для мобильных

### 1. **Touch-friendly Interface**
- Минимальные размеры кнопок: 44-48px
- Достаточное расстояние между интерактивными элементами
- Оптимизированный размер текста (min 16px для input)

### 2. **Safe Area Support**
- Поддержка notched устройств (iPhone X+)
- Класс `.safe-area-inset-top` для поддержки верхней полосы
- Класс `.pb-safe` для поддержки нижней полосы

### 3. **Lazy Loading**
- Добавлена поддержка lazy loading для изображений
- Placeholder стили для лучшего UX

### 4. **Accessibility**
- Улучшена поддержка клавиатурной навигации
- Добавлены aria-labels для интерактивных элементов
- Поддержка prefers-reduced-motion для доступности

### 5. **Performance**
- Оптимизированы font loads
- DNS Prefetch для внешних ресурсов
- Preconnect для внешних ресурсов
- Content-visibility для изображений

## 📝 Класс-помощники

### Touch Targets
```css
.touch-target {
  min-height: 44px;
  min-width: 44px;
  display: flex;
  align-items: center;
  justify-content: center;
}
```

### Safe Area Support
```css
@supports (padding: max(0px)) {
  .safe-area-inset-top {
    padding-top: max(0px, env(safe-area-inset-top));
  }
}
```

### Responsive Text
```css
.text-responsive-lg {
  font-size: clamp(1.125rem, 5vw, 1.875rem);
}
```
